## 1\. 🔍 프롬프트 유형과 선택 이유

| 구분 | 선택한 유형 | 선택 이유 |
| :--- | :--- | :--- |
| **프롬프트 유형** | **페르소나 지정 (Persona Prompting)** & **매체 변환 및 출력 제약 (Media Transformation & Constraint)** | 복잡한 전문 자료를 **영상 문법**과 **쉬운 언어**에 맞게 재구성하는 능력을 평가합니다. |
| **선택 이유** | **페르소나 지정:** '숏폼 PD' 역할을 통해 60초 내외의 짧은 호흡, '초등학생 눈높이'의 친근한 톤을 강제합니다. |
| | **매체 변환 및 제약:** **60초 분량**, **후킹 멘트**, **시각/음향 요소**, **JSON 형식**이라는 엄격한 제약을 걸어 변환 및 구조화 능력을 극대화합니다. |

-----

## 2\. 📝 작성 위해 추가 필요 요건

AI가 전문 리포트 내용을 '60초 숏폼 대본'으로 변환하기 위해, **가정된 리포트의 핵심 내용**이 필요합니다. (PDF 링크 분석 결과 기반)

  * **리포트 핵심 요약:**
      * **미래:** 멀티모달 기술 진화, 콘텐츠 제작 비용 90% 이상 절감.
      * **윤리 쟁점:** 저작권 침해, 할루시네이션(거짓 정보), 편향성.
      * **결론:** '통제 가능한 AI'를 위한 규제와 기업의 'Human-in-the-loop(인간 개입)' 최종 검수 필요.
  * **변환 목표:** 전문 용어를 초등학생도 이해하는 수준으로 변환할 것.

-----

## 3\. ⌨️ 문제 해결 프롬프트

````prompt
당신은 뉴미디어 스타트업 '테크인사이드'의 '숏폼 콘텐츠 PD'입니다. 당신의 목표는 전문적인 '생성형 AI의 미래와 윤리적 쟁점' 리포트 내용을, **'초등학생도 이해하는 수준'**의 언어로 바꾸고, **60초 분량**의 숏폼 영상 대본으로 변환하는 것입니다.

**[핵심 제약 조건]**
1.  **분량:** 총 러닝타임 60초 내외로 5~6개 씬으로 구성하십시오.
2.  **톤앤매너:** 친근하고 흥미를 유발하는 뉴미디어 스타일 (쉬운 용어 사용 필수).
3.  **필수 요소:** 초반 3초 안에 시선을 사로잡는 강력한 **'후킹(Hooking)' 멘트**와 각 씬별 **'visual_cue' (시각적 요소)**, **'sound_effect' (효과음/배경음악)**를 명시하십시오.

**[리포트 핵심 요약]**
* **미래:** AI는 창작을 돕고 제작 비용을 획기적으로 절감함.
* **윤리 쟁점:** 저작권 침해, 할루시네이션(거짓말), 데이터 편향성 문제 발생.
* **결론:** 'Human-in-the-loop' (인간의 최종 검수)와 규제 마련이 필요함.

**[출력 형식]**
```json
{
  "title": "60초 지식 요약: 초딩도 이해하는 'AI가 그린 그림'의 비밀",
  "persona": "숏폼 콘텐츠 PD",
  "running_time_sec": 60,
  "scenes": [
    {
      "time_code": "00:00 - 00:03",
      "script": "여기에 후킹 멘트 입력",
      "visual_cue": "시각적 요소/화면 구성",
      "sound_effect": "효과음 및 배경음악"
    },
    // ... 나머지 씬들
  ]
}
````
