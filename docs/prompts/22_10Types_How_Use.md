| 구분 | 선택한 유형 | 선택 이유 |
| :--- | :--- | :--- |
| **프롬프트 유형** | **목표 기반 코드 생성 (Goal-Oriented Code Generation)** & **구조/스타일 제약 (Structure & Style Constraint)** | 핵심 문제를 해결하기 위해, 다자간의 상충되는 요구사항(직관성, 다크 모드, 데이터 밀집)을 단일 파일 코드 생성으로 즉시 시각화하도록 유도합니다. |

## 2\. 📝 작성 위해 추가 필요 요건

AI가 웹 프로토타입을 생성하기 위해, 상충되는 이해관계자들의 핵심 요구사항을 기술적 요소로 변환하여 프롬프트에 포함해야 합니다.

  * **디자인팀 요구 (트렌디한 다크 모드):**
      * **스타일:** 다크 모드(Dark Mode) 적용.
      * **프레임워크:** Tailwind CSS 사용 (신속한 스타일링).
  * **운영팀 요구 (데이터 밀집형 테이블):**
      * **기능:** 타임스탬프 기반 댓글 목록을 테이블 형태로 표시.
      * **기능:** 댓글에 상태(반영/미반영) 표시 및 변경 버튼 제공.
  * **기획팀 요구 (직관적인 UI):**
      * **배치:** 중앙에 영상 플레이어, 오른쪽에 승인/댓글 영역 배치 (직관적 흐름).
      * **기능:** 상단에 최종 '승인' 및 '반려' 버튼 배치.

## 3\. ⌨️ 문제 해결 프롬프트

> 이 프롬프트는 3년 차 개발자가 급박한 회의 전까지 프로토타입을 확보하기 위해 AI에게 입력하는 내용입니다.

```prompt
당신은 '네오스트림'의 프론트엔드 개발자입니다. 다음 요구사항들을 모두 충족하는 단일 파일 웹 페이지 프로토타입을 HTML로 즉시 작성하십시오.

**[코딩 제약 조건]**
1.  **단일 파일:** 모든 HTML, Tailwind CSS, JavaScript는 하나의 파일(`review_dashboard.html`) 내에 포함되어야 합니다.
2.  **스타일링:** 트렌디한 **다크 모드** 디자인을 적용하고, 스타일링은 **Tailwind CSS**를 사용하여 처리하십시오.
3.  **영상 플레이어:** 중앙에 크게 배치하고, 더미 영상 URL을 사용하십시오.

**[필수 UI/UX 요소]**
1.  **상단 컨트롤:** 페이지 최상단에 큰 '최종 승인' 및 '반려' 버튼을 배치합니다. (빨간색/초록색으로 명확히 구분)
2.  **댓글 테이블:** 영상 옆(혹은 하단)에 **데이터 밀집형** 디자인의 '타임스탬프 기반 댓글 목록' 테이블을 만드십시오.
    * 컬럼: `ID`, `시간 (Time)`, `내용 (Content)`, `상태 (Status)`, `액션 (Action)`
    * 더미 데이터 3~4개를 포함하십시오.
3.  **인터랙션:** 각 댓글 행에 '상태 반영' 토글 버튼(혹은 일반 버튼)을 추가하여, 상태를 '미반영'에서 '반영 완료'로 변경할 수 있도록 JavaScript 로직을 구현하십시오. (상태 변경 시 UI 업데이트 필수)
4.  **폰트:** Inter 폰트를 사용하십시오.
```
