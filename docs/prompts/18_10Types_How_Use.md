## 1\. 🔍 프롬프트 유형과 선택 이유

| 구분 | 선택한 유형 | 선택 이유 |
| :--- | :--- | :--- |
| **프롬프트 유형** | **페르소나 지정 (Persona Prompting)** & **Chain-of-Thought (CoT) 결합** | **위기 상황 대응 능력**과 **깊이 있는 분석**을 동시에 확보하기 위해 선택했습니다. |
| **선택 이유** | **페르소나 지정:** '광고 대행사 콘텐츠 제작 팀장' 역할을 부여하여, 광고 성과 지표(CTR, 전환율, 이탈률)에 대한 전문적인 이해를 바탕으로 분석하도록 유도합니다. |
| | **Chain-of-Thought (CoT):** 문제를 '현상 진단-원인 가설 수립-긴급 해결책 제시'의 3단계 논리 구조로 나누어, 체계적이고 즉각적인 대응 방안을 도출하도록 강제합니다. |

## 2\. 📝 작성 위해 추가 필요 요건

프롬프트의 결과물인 **긴급 대응 방안**의 실효성을 높이고, C 영상의 실패 원인을 구체적으로 분석하기 위해 다음과 같은 핵심 정보를 추가하여 AI에게 제공해야 합니다.

1.  **C 영상의 구체적인 구성 (Input):**
      * **C 영상(6초)의 타임라인별 구성:** (예: 0\~1초: 제품 이미지, 1\~4초: '50% 할인' 문구, 4\~6초: '지금 구매' CTA 및 가격 노출).
      * **광고 타겟:** C 영상만 다른 타겟층에 노출되었는지 (예: A/B는 뷰티 관심사, C는 광범위한 할인 쇼핑몰 방문자).
2.  **클라이언트 'A' 브랜드의 특성 (Context):**
      * **브랜드의 포지셔닝:** (예: '프리미엄 고가' 브랜드 vs. '가성비' 브랜드). 할인 혜택이 브랜드 이미지와 충돌하지 않았는지 판단의 근거가 됩니다.

## 3\. 📝 문제 해결 프롬프트

### **문제 해결 프롬프트 (Persona + CoT 적용)**

```prompt
당신은 광고 대행사의 '콘텐츠 제작 팀장'입니다. 아래 긴급한 유튜브 범퍼 광고 성과 데이터를 분석하고, '현상 진단-원인 가설 수립-긴급 대응 방안'의 3단계 Chain-of-Thought 구조로 즉각적인 보고서를 작성하십시오.

**[문제 상황 및 성과 데이터]**
* **매체:** 유튜브 범퍼 광고 (6초).
* **문제 영상(C):** '할인 혜택 강조' 버전.
* **성과 지표:** CTR 0.8% (A/B 영상 2.5% 대비 심각하게 낮음), 구매 전환율 0.1%.
* **이탈률:** 3초 이내 이탈률 80% (매우 높음).
* **예산 투입:** 3개 영상 중 가장 많은 예산 투입.

**[추가 데이터 (가정)]**
1.  **C 영상의 구성 (6초):** 0~2초: '파격 할인' 문구와 가격표 노출, 2~5초: 제품이 아닌 **일반적인 쇼핑 화면** 및 '선착순' 문구, 5~6초: CTA ('지금 구매').
2.  **타겟 페르소나:** A, B 영상은 뷰티 관심사 타겟, C 영상은 **광범위한 '가격 민감형 쇼퍼' 타겟**.
3.  **브랜드 포지셔닝:** 클라이언트 'A'는 **프리미엄, 고기능성**을 강조하는 뷰티 브랜드.

**[요청 사항]**
1.  **현상 진단 (Diagnosis):** 데이터 기반으로 C 영상의 실패 원인이 **'어떤 지표의 어떤 문제'**인지 핵심을 짚으십시오.
2.  **원인 가설 수립 (Causal Hypothesis):** 3초 이내 이탈률 80%의 핵심 원인을 광고 구성 및 타겟팅 관점에서 **세 가지 핵심 가설**로 분석하십시오.
3.  **긴급 대응 방안 (Emergency Solution):** 클라이언트에게 보고할 수 있도록, C 영상을 **즉시 수정(Re-edit)** 또는 **대체**할 수 있는 **구체적인 2가지 액션 플랜**을 제시하십시오.
```

## 4\. 🚀 프롬프트 실행 결과 (예상)

### **[프롬프트 실행 결과]**

#### **1. 현상 진단 (Diagnosis)**

> **핵심 진단:** C 영상은 \*\*초기 3초 내 이탈률 80%\*\*로 인해 노출 대비 클릭(CTR 0.8%) 자체가 이루어지지 않았으며, **'할인' 소구점**이 프리미엄 브랜드 이미지와 **타겟팅 실패**를 일으켜 구매 전환율(0.1%)까지 붕괴된 **'광고의 구조적 결함'** 문제입니다.

#### **2. 원인 가설 수립 (Causal Hypothesis)**

  * **가설 1: '할인' 소구점의 브랜드 이미지 충돌 (Conversion Failure):** 프리미엄을 지향하는 브랜드가 6초 광고의 대부분을 '할인 혜택'과 '선착순'에 할애함으로써, 타겟에게 **저가 브랜드**로 오인되게 만들었습니다. 이는 이미 제품의 가치(A, B 영상)에 매료된 잠재 고객의 **구매 의욕을 상실**시켜 전환율을 파괴했습니다.
  * **가설 2: 제품 정보의 초기 부재 (Attention Failure):** 3초 이내 이탈률 80%는 시청자들이 **자신과 관련 없는 광고**라고 즉시 판단했다는 증거입니다. 0\~2초에 '할인 문구'만 먼저 노출하고 **정작 핵심 제품 이미지가 늦게 등장**하거나 **미약**하여, 시청자가 '이게 무슨 제품의 할인이지?'라는 의문을 해소하지 못하고 즉시 스킵했습니다.
  * **가설 3: 광범위 타겟팅의 비효율 (Targeting Mismatch):** C 영상만 '가격 민감형 쇼퍼'라는 광범위 타겟에 노출되면서, **실제 뷰티 신제품에 관심 없는** 다수의 시청자들에게 노출되었습니다. 이들은 혜택을 봐도 클릭하지 않았고, 클릭하더라도 제품의 필요성을 못 느껴 전환율이 극도로 낮아졌습니다.

#### **3. 긴급 대응 방안 (Emergency Solution)**

| 구분 | 액션 플랜 | 구체적인 실행 계획 |
| :--- | :--- | :--- |
| **액션 1** | **C 영상 즉시 수정 (Re-edit)** | **6초 구성 재배치:** 1초: **제품/모델의 가장 후킹한 비주얼** 노출 (A/B 영상의 성공 요소 차용). 2\~4초: **가장 큰 혜택 숫자**를 빠르게 노출. 5\~6초: 제품명과 **CTA**. (할인 소구점을 뒤로 밀고, 제품의 시각적 매력을 전면 배치하여 3초 이탈률 최소화). |
| **액션 2** | **C 영상 타겟 즉시 재설정** | **타겟팅 변경:** C 영상의 예산을 A, B 영상의 **성공적인 뷰티 관심사 타겟**에게 집중 투입합니다. '할인 혜택'을 이미 제품에 관심 있는 타겟에게 \*\*'구매 결정의 촉매제'\*\*로 활용하여 CTR을 높이고 전환율을 방어합니다. |
| **추가 조치** | **예산 재분배** | C 영상의 남은 예산 중 70%는 CTR/전환율이 입증된 A, B 영상에 **즉시 재분배**하여 캠페인 성과 전체의 하락을 방어하고, 30%는 수정된 C 영상(액션 1)에 제한적으로 재집행하여 효과를 검증합니다. |
