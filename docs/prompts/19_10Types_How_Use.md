## 1\. 🔍 프롬프트 유형과 선택 이유

| 구분 | 선택한 유형 | 선택 이유 |
| :--- | :--- | :--- |
| **프롬프트 유형** | **페르소나 지정 (Persona Prompting)** & **출력 형식 지정 (Format Constraint) 결합** | \*\*창의적 사고(Creative Director 역할)\*\*를 극대화하면서, 동시에 **실무에서 바로 사용 가능한** 구체적인 기획안 형태로 결과물을 도출하도록 유도합니다. |
| **선택 이유** | '크리에이티브 디렉터' 페르소나는 **MZ세대 타겟의 눈높이**에 맞춰 복잡한 기술 용어를 감각적인 광고 컨셉으로 변환하는 **창의적인 역할**을 수행하게 합니다. '3가지 시나리오', '30초 분량' 등의 명확한 제약 조건은 아이디어가 구체적이고 실현 가능하도록 합니다. |

## 2\. 📝 작성 위해 추가 필요 요건

| 필요 요건 | 기술 명세서에서 추출된 핵심 정보 |
| :--- | :--- |
| **제품의 핵심 효능 3가지** | 1. **80% 시간 절감** (반복 업무 자동화) |
| | 2. **코딩 없이 5분 내 워크플로우 구성** (노코드/로우코드) |
| | 3. **기존 업무 툴과 완벽 통합** (Slack, Gmail, Notion, Jira 등) |
| **MZ세대 타겟 공감 코드** | '노가다 업무' 해방, '칼퇴', '워라밸' |

## 3\. 📝 문제 해결 프롬프트

### **문제 해결 프롬프트 (Persona + Format Constraint 적용)**

```prompt
당신은 '광고 대행사의 크리에이티브 디렉터'입니다. 아래 클라이언트 'A'의 AI 기반 업무 자동화 SaaS 기술 명세서를 분석하여, 복잡한 기능을 MZ세대 직장인 타겟의 흥미를 유발할 30초 숏폼(릴스/틱톡) 광고 시나리오 3가지로 변환해야 합니다. 당신의 목표는 "기술적 복잡성"을 해소하고 "실질적인 삶의 변화"를 보여주는 것입니다.

**[핵심 입력 데이터 (기술 명세서 기반)]**
* **제품 특징:** 반복적인 업무 프로세스를 AI 기반으로 자동화하는 노코드/로우코드 솔루션입니다.
* **가장 강력한 기능 3가지:** 1.  **반복 업무 자동화로 80% 시간 절감**
    2.  **코딩 없이 5분 내 워크플로우 구성** (드래그 앤 드롭 인터페이스)
    3.  **Slack, Gmail, Notion, Jira 등 툴 완벽 통합**
* **타겟 페인 포인트:** 끝없는 '노가다 업무', 복잡한 기술 용어, 야근.
* **궁극적 혜택:** '저녁이 있는 삶(칼퇴)', '높은 업무 효율성(갓생)'.

**[요청 사항]**
아래 표 형식에 맞춰, 30초 분량의 **각기 다른 핵심 소구점**을 가진 숏폼 시나리오 3가지를 구체적으로 도출하십시오.

| 시나리오 # | 핵심 소구점 | 핵심 카피 (Headline) | 비주얼 컨셉 및 구성 (30초) | 활용 BGM 아이디어 |
| :--- | :--- | :--- | :--- | :--- |
| **1.** | **'노가다' 없는 칼퇴 (80% 시간 절감)** | (MZ세대 공감형 카피) | (반복, 가속화, 대비를 통한 유머/공감형 상황극) | (트렌디한 밈 BGM) |
| **2.** | **코딩 없는 심플함 (노코드)** | (기술 용어 탈피 카피) | (직관적 조작, 화면 전환의 편리성 강조) | (빠르고 경쾌한 K-POP) |
| **3.** | **툴 간의 완벽 통합** | (연결성/동기화 강조 카피) | (멀티 스크린 구성, 데이터의 흐름 시각화) | (드라마틱한 영화 OST) |

## 4. 🚀 프롬프트 실행 결과 (예상)

### **[프롬프트 실행 결과]**

| 시나리오 # | 핵심 소구점 | 핵심 카피 (Headline) | 비주얼 컨셉 및 구성 (30초) | 활용 BGM 아이디어 |
| :--- | :--- | :--- | :--- | :--- |
| **1.** | **'노가다' 없는 칼퇴** | **"업무 속도 80% 올리고, 칼퇴 속도는 100배 올렸다."** | **[1~10초] 반복:** 주인공이 챗봇 응답 분류, 보고서 정리 등 반복 업무에 지쳐 손이 느려지는 모습 슬로우 모션 연출. **[10~25초] 가속:** AI SaaS를 켜자 업무가 **80% 가속**되는 화면 오버랩. **[25~30초] 대비:** 주인공이 "나 먼저 퇴근!"을 외치며 힙하게 회사 문을 나서는 장면을 슬로우 모션으로 보여줌. | **BGM:** 'Calm & Chill' 계열의 힙합 비트 (유명 틱톡 챌린지 배경음악) |
| **2.** | **코딩 없는 심플함** | **"나 코딩 못함. 근데 워크플로우는 5분 만에 만듦."** | **[1~10초] 문제 제기:** 주인공이 노트북 앞에서 Python 코드나 복잡한 API 명세서를 보며 좌절하는 모습. "코딩 지식 없음" 자막 노출. **[10~25초] 솔루션:** AI SaaS의 **드래그 앤 드롭 비주얼 에디터**를 통해 5분 만에 워크플로우를 완성하는 손짓 시연. **[25~30초] 결과:** "노코드/로우코드 솔루션입니다" 자막과 함께 완성된 워크플로우가 돌아가는 모습. | **BGM:** 밝고 경쾌하며 템포가 빠른 K-POP. (성공, 성취감 관련 노래) |
| **3.** | **툴 간의 완벽 통합** | **"이메일, 슬랙, 노션 데이터가 한 팀인 비결."** | **[1~10초] 혼란:** Gmail, Slack, Notion 화면이 정신없이 전환되며 업무 흐름이 끊기는 모습. **[10~25초] 통합:** AI SaaS 화면을 중심으로 Gmail의 이메일 분류 결과가 **자동으로 Notion 보고서로 정리**되고, 완료 알림이 **Slack으로 전송**되는 데이터의 흐름을 그래픽으로 시각화. **[25~30초] 깔끔함:** 모든 업무가 하나의 대시보드 안에서 정리된 화면. | **BGM:** 드라마틱하고 웅장한 영화 OST (미션 임파서블, 어벤져스 스타일) |
```
